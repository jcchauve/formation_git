Itération 15 : Supprimer un tag

Ajouter un tag

Nous allons créer un tag sur le commit précédent celui de la v1.

Executer

git tag v1-beta v1^
git show -s v1-beta
git hist

Sortie standard

$ git tag v1-beta v1^

$ git show -s v1-beta
commit 5a67ff57942cc177b3bfacc076e872a8d8d377a3
Author: Steven Enten <steven@enten.fr>
Date:   Sat Mar 21 17:28:29 2015 +0100

    Ajout fonctionnalité personnification

$ git hist
* 8b88904 2015-03-21 | Création carnet d'adresses développeurs (HEAD, tag: v1, master) [Steven Enten]
* 5a67ff5 2015-03-21 | Ajout fonctionnalité personnification (tag: v1-beta) [Steven Enten]
* 421a6a8 2015-03-21 | Initialisation du projet (tag: v0) [Steven Enten]

Le caractère ^ après une référence permet de désigner son parent. Le tag v1-beta ne pointe pas sur le commit de v1 mais de son parent.

Retirer un tag

L'option -d de la commande git tag permet de supprimer un tag.

Nous souhaitons supprimer le tag v1-beta.

Executer

git tag
git tag -d v1-beta
git tag

Sortie standard

$ git tag
v0
v1
v1-beta

$ git tag -d v1-beta
Deleted tag 'v1-beta' (was 5a67ff5)

$ git tag
v0
v1
