Itération 4 : Vérifier l'état du projet

Vérifier le statut du dépôt

Nous allons utiliser la commande git status pour vérifier l'état actuelle de notre dépôt.

Exécuter

git status

Sortie standard

$ git status
On branch master
nothing to commit, working directory clean

Cette vérification nous informe que l'état de notre répertoire de travail est historiser : il n'y a pas de modifications à valider dans l'historique. Notre base Git est à jour.

La commande git status est régulièrement utilisée pour contrôler l'état du dépôt, de la zone index et du répertoire de travail.
