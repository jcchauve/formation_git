Itération 17 : Créer une branche

Créer une nouvelle branche

Nous prévoyons d'ajouter une nouvelle fonctionnalité à notre projet. Son développement aura une certaine durée dans le temps. Son aspect expérimentale nous contraint à vouloir séparer son développement de la branche de développement principale master.

Nous allons utiliser la commande git branch pour créer une branche dans laquelle nous développerons notre nouvelle fonctionnalité.

Si son développement abouti à un résultat satisfaisant, nous fusionnerons ses modifications à la branche principale. Sinon nous pourrons la supprimer sans que cela ai un impact sur la branche master.

Executer

git branch
git branch experimental
git branch

Sortie standard

$ git branch
* master

$ git branch experimental

$ git branch
  experimental
* master

La nouvelle branche experimental a bien été créée. Mais c'est toujours la branche principale master qui est active. Il faut utiliser la commande git checkout pour changer de branche.
