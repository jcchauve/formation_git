Itération 22 : Supprimer des branches

Supprimer une branche

Si la fusion de la branche master avec la branche experimental s'est bien passée, nous n'avons plus besoin de la branche experimental. Nous allons la supprimer.

La commande git branch avec l'option -d permet de supprimer une branche.

Executer

git branch -v
git branch -d experimental
git branch -v

Sortie standard

$ git branch -v
  experimental 3c75580 Init doc find-dev.sh
* master       7f5dcf7 Fix conflit dans README.md

$ git branch -d experimental
Deleted branch maintainers-dev (was 3c75580).

$ git branch -v
* master 7f5dcf7 Fix conflit dans README.md

Notre branche experimental a bien été supprimée. Sa suppression ne supprime pas l'historique de ses commits.
