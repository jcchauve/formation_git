Itération 27 : Tirer des modifications

Définition

La commande git pull est un moyen rapide de récupérer et d'intégrer les modifications des branches distantes.

Retenez simplement que git pull équivaut à exécuter les deux commandes suivantes.

git fetch origin
git merge origin/master

Le concept de "tirer" des modifications ou de pull consiste à récupérer les modifications distantes et à les fusionner à la branche locale de suivi.
