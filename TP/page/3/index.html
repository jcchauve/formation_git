Itération 3 : Créer un projet

Créer un projet "Hello, World"

Nous allons commencer par créer un nouveau répertoire de travail vide nommé hello. Ce répertoire contiendra le code de notre projet et l'historique des modifications.

Notre projet consiste en un script Shell hello.sh affichant le message Hello, World dans le terminal.

Nous allons initialiser le code de notre projet.

Exécuter

cd ~
pwd
mkdir hello
cd hello
cat > hello.sh <<EOF
#!/bin/bash
echo "Hello, World"
exit 0
EOF
chmod +x hello.sh
./hello.sh

Exécuter

$ cd ~

$ pwd
C:\Users\Steven

$ mkdir hello

$ cd hello

$ cat > hello.sh <<EOF
> #!/bin/bash
> echo "Hello, World"
> exit 0
> EOF

$ chmod +x hello.sh

$ ./hello.sh
Hello, World

Le caractère ~ est un alias vers le répertoire de l'utilisateur. La commande cd ~ ouvre le répertoire de l'utilisateur courant.

  • Sous Windows, ~ désigne le répertoire C:\Users\Moi\
  • Sous Linux, ~ désigne le répertoire /home/moi/

Remplacer Moi par le login de votre compte utilisateur

Créer un dépôt Git

Nous allons maintenant utiliser la commande git init pour initialiser l'historique de notre répertoire de travail hello. Nous pourrons ainsi versionner les fichiers de notre projet.

Exécuter

git init

Sortie standard

$ git init
Initialized empty Git repository in /home/moi/hello/.git/

Historiser dans le dépôt

Nous allons enregistrer l'état de notre projet pour créer un premier état validé dans l'historique des révisions.

Exécuter

git add hello.sh
git commit -m "Initialisation du projet"

Sortie standard

$ git add hello.sh

$ git commit -m "Premier commit"
[master (commit racine) 421a6a8] Initialisation du projet
 1 file changed, 3 insertions(+)
 create mode 100755 hello.sh
